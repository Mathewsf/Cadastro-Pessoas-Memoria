# Cadastro de Pessoas em Memória

Projeto desenvolvido em linguagem C para a disciplina de Técnicas de Programação.

## Funcionalidades

- Inserção de pessoas
- Busca por CPF
- Busca por Nome
- Listagem por ordem de inserção
- Listagem por CPF
- Listagem por Nome
- Exclusão lógica
- Exclusão física

## Estruturas Utilizadas

- Struct Pessoa
- Vetor de Pessoas
- Vetor Índice CPF
- Vetor Índice Nome

## Algoritmos Utilizados

- Busca Binária
- Inserção Ordenada

## Observação

Nesta primeira versão todos os dados permanecem apenas na memória RAM durante a execução do programa.
Ao encerrar o programa, os dados são perdidos.
